Distant Lights

Physics-informed light-to-sound demo

Hear modeled electrical hum, a simplified photoacoustic response, and carrier-based sonification for slow flashing lights. This version is a static site: no build step, no framework install, no blank-screen GitHub Pages issue.

How to use this demo

  1. Choose a preset or edit the sliders.
  2. Press Play to hear the modeled output.
  3. Watch the waveform previews to see the source and output signals.
  4. Download WAV for high-quality audio, or Record 8s for a quick live capture.
  5. Use the theme tab to try colours, then keep the final palette for the next build.
